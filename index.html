模擬試験の概要

建築設備士第二次試験（設計製図）を想定し、設備計画の条件整理から記述・図面作成までを段階的に進めるための土台です。

想定時間 180分
用紙 A4印刷対応
形式 HTML / CSS / JavaScript

計画条件

今後、課題データをdataディレクトリに追加し、用途・延床面積・設備条件などを表示できるように拡張します。

  • 建築概要
  • 空気調和設備
  • 給排水衛生設備
  • 電気設備

Generator Preview

Building GeneratorとEquipment Generatorを順番に実行し、生成された模擬試験条件をブラウザで確認できます。

「模擬試験生成」ボタンを押すと、建築条件と設備条件がカード形式で表示されます。

生成JSON

「JSONを表示」ボタンを押すと、生成済みの建築条件と設備条件をJSON形式で確認できます。

SVG基盤プレビュー

A3横の編集可能なSVG図面基盤を表示し、図枠・タイトル枠・基本記号を確認できます。

「SVGサンプル表示」ボタンを押すとA3横のサンプル図面を表示します。

建築図プレビュー

Drawing GeneratorのJSONから、建築基本設計図風のA3横SVG（配置図・各階平面図・白図）を生成します。

模擬試験生成後に建築図を表示できます。

設備図プレビュー

建築図SVGを背景に、空調・換気、給排水衛生、電気設備レイヤーを重ねて表示します。

模擬試験生成後に設備図を表示できます。

試験問題プレビュー

PlannerからDrawing Generatorまでの生成結果をもとに、建築設備士第二次試験（設計製図）形式の問題集を生成します。

「試験問題を生成」ボタンを押すと問題集データを作成します。

試験問題JSONはここに表示されます。

答案用紙プレビュー

Exam/Material/Drawing Generatorの結果から、空欄中心のA4縦・A3横答案用紙を生成します。

「答案用紙を生成」ボタンで答案用紙セットを作成します。

印刷

印刷時はメニューや操作ボタンを非表示にし、白背景のA4レイアウトに最適化します。
