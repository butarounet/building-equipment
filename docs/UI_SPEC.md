
## SVG基盤プレビューUI

トップページに「SVG基盤プレビュー」セクションを追加します。「SVGサンプル表示」ボタンでA3横の図枠、タイトル枠、方位記号、グリッド線、壁、柱、扉、窓、室名、寸法線、縮尺バーを含むサンプルSVGを描画します。「SVG保存」ボタンは生成済みSVGを`.svg`ファイルとして保存します。未生成時に保存を押した場合は、先にサンプル表示を行うようメッセージを表示します。

画面表示ではSVGをプレビュー領域内に縮小表示し、印刷時はA3横の原寸比率を維持します。既存のGenerator PreviewおよびJSON表示は独立して動作し、SVGプレビュー操作によって生成済みJSON表示は変更されません。

## 建築図プレビューUI

`index.html`に「建築図プレビュー」セクションを設ける。選択項目は配置図、地下1階、1階、2階、3階、代表階、塔屋、屋上、白図とし、操作は「建築図を表示」「SVG保存」「印刷」とする。既存のGenerator Preview、JSON表示、SVG基盤プレビューは維持する。

未生成時は「模擬試験が未生成」、Drawing Generatorデータがない場合は「Drawing Generatorのデータがありません」、階がない場合は選択階名を含むエラー、室データ不足時は「室データが不足しています」、SVG生成失敗時は失敗理由をプレビュー領域に表示する。

印刷時はA3横、余白0、背景白、操作ボタン非表示、図面のみ表示、SVG縦横比維持、黒線・判読可能文字を基本とする。
