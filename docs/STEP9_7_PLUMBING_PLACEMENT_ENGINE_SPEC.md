# Step9-7 給排水衛生設備配置エンジン

## 目的

Step9-5で生成した設備スペースとStep9-4の平面計画を入力として、建築設備士第二次試験（設計製図）レベルで成立する給排水衛生設備配置案を自動生成する。

対象は給水、給湯、排水、通気、雨水、消火、排水槽、受水槽、高置水槽、雑用水、ポンプ設備である。

## 入力

- `floorPlan`: FloorPlan JSON。階、室、コア、シャフト候補を参照する。
- `equipmentSpace`: EquipmentSpace JSON。未指定時はStep9-5 Plannerで補完する。
- `rooms` / `roomList`: 追加の室リスト。
- `corePosition`: コア位置。
- `shaftNetwork`: PS・EPS・DSの上下階整合情報。
- `buildingUse`: 建物用途。
- `plumbingConditions`: 日使用水量係数などの給排水条件。
- `fireProtectionConditions`: 消火設備条件。

## 処理フロー

1. Room Fixture Analysis: 室名、ゾーン、面積から衛生器具数、給水径、排水径、通気径、給湯要否を推定する。
2. System Selection: 飲料水系統、雑用水系統、中央/局所給湯、汚水・雑排水・厨房排水、消火方式を選定する。
3. 立管配置: Step9-5のPSを優先し、給水立管、排水立管、通気立管、雨水立管を上下階整合させる。
4. 横主管配置: 受水槽、高置水槽、ポンプ室、排水槽、消火ポンプ室を接続する主配管を配置する。
5. 枝管配置: 各水回り室の中心へ給水・給湯・排水・通気枝管を直交折れ線で接続する。
6. 器具接続: 客室、便所、浴室、厨房、レストラン、宴会場などの器具を配管ネットワークへ接続する。
7. 排水勾配チェック: 排水枝管に管径別の標準勾配を設定し、成立可否を判定する。
8. 圧力損失チェック: 配管延長をもとに給水・給湯の圧力損失リスクを評価する。
9. 設備干渉チェック: 長大配管を低リスク干渉候補として抽出し、Step9-9へ渡す。
10. Score: 100点満点で配管長、上下階整合、施工性、保守性、更新性、法適合、圧力損失、排水勾配、防災を評価する。

## 判定ロジック

- PS優先: 給水、給湯、排水、通気はPS中心点を優先利用する。
- 水回り抽出: 客室、便所、浴室、洗面、厨房、レストラン、宴会用途を器具接続対象とする。
- 給湯方式: 給湯対象が一定数を超える場合は中央給湯、少数の場合は局所給湯とする。
- 排水分類: 厨房は厨房排水、客室は汚水・雑排水、その他水回りは雑排水とする。
- 雨水処理: 各階外形からルーフドレン、雨水立管、雨水利用フラグを生成する。
- 消火設備: 各階PS近傍に屋内消火栓、主要室にスプリンクラーヘッドを配置する。
- 評価: 配管総延長、排水勾配、上下階整合、保守性、法適合、防災を加点し、長大ルートを減点する。

## Engine構成

`js/planner/plumbingPlacementEngine.js` は以下のAPIを公開する。

- `planPlumbing(input)`: Plumbing JSON全体を生成する。
- `WaterSupplyPlanner`: 飲料水系統、雑用水系統、受水槽、高置水槽、揚水ポンプ、給水立管、給水枝管を計画する。
- `HotWaterPlanner`: 中央給湯、局所給湯、給湯配管、循環配管、膨張タンク、給湯ポンプを計画する。
- `DrainagePlanner`: 汚水、雑排水、厨房排水、特殊排水、排水槽、排水ポンプを計画する。
- `VentPipePlanner`: 通気立管、伸頂通気、逃がし通気、結合通気、ループ通気を計画する。
- `StormDrainPlanner`: 屋上雨水、ルーフドレン、雨水立管、雨水桝、浸透設備、雨水利用を計画する。
- `FireProtectionPlanner`: 屋内消火栓、スプリンクラー、補助散水栓、消火配管、消火ポンプを計画する。
- `PipeRoutingEngine`: 給水、給湯、排水、通気、雨水、消火の配管ルートを探索する。
- `PlumbingScorer`: 100点満点で給排水衛生設備計画を評価する。

## 出力

- `fixtureAnalysis`: 室別器具分析。
- `waterSupply`: 受水槽、高置水槽、揚水ポンプ、給水立管、給水枝管。
- `hotWater`: 給湯方式、給湯機、膨張タンク、給湯ポンプ、給湯配管。
- `drainage`: 排水立管、排水枝管、排水槽、排水ポンプ。
- `vent`: 通気立管、通気枝管、結合通気。
- `storm`: ルーフドレン、雨水立管、雨水利用。
- `fireProtection`: 消火ポンプ、屋内消火栓、スプリンクラー。
- `pipeNetwork`: 後続Step9-8、Step9-9、Step9-10へ入力可能な共通配管ネットワーク。
- `equipmentLayout`: 機器配置図用の機器矩形一覧。
- `pipeSchedule`: 配管一覧。
- `ascii`: 給水系統図、排水系統図、通気系統図、消火系統図。
- `improvements`: 改善提案。
- `score`: 100点満点の評価。
- `checklist`: Quality Checker結果。
