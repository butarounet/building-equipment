# 建築設備士 第二次試験 模擬試験

建築設備士第二次試験（設計製図）の本試験レベルの模擬試験を作成するためのWebアプリです。

## 構成

```text
/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── app.js
├── svg/
├── data/
├── docs/
└── README.md
```

## 機能

- 白背景のシンプルな学習画面
- ヘッダーとメニュー
- レスポンシブ対応
- A4縦印刷対応
- CSSとJavaScriptの分離

## 使い方

`index.html` をブラウザで開いて利用します。印刷する場合は画面内の「A4で印刷する」ボタン、またはブラウザの印刷機能を使用してください。
